### Jimbo-v2ray Scanner
Jimbo-v2ray is a multithread v2ray configs scrapper, chunk base and AI driven project that automatically crawl the web links \
in order to find fresh configs and scan for real delay and sing-box download test on your network.

![alt text](https://raw.githubusercontent.com/mahdizynali/jimbo-v2ray/refs/heads/main/assets/result.png)

## Notice
This version is developing over linux debian distribution but it works with windows too!

---

### Optional Sing-Box
For testing under sing-box you may install it via official github page :
```
wget https://github.com/SagerNet/sing-box/releases/download/v1.12.20/sing-box_1.12.20_linux_amd64.deb
```
```
sudo dpkg -i sing-box_1.12.20_linux_amd64.deb
```

---

### Run App
```
python3 app.py
```
