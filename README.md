### Jimbo-v2ray Scanner
Jimbo-v2ray is a v2ray configs scrapper that automatically crawl the web links \
in order to find fresh configs and scan for real delay and sing-box download test on your network.

## Notice
This version is developing over linux debian versions

---

### Optional Sing-Box
For testing under sing-box you may install it via official github page :
```
wget https://github.com/SagerNet/sing-box/releases/download/v1.12.20/sing-box_1.12.20_linux_amd64.deb
```
```
sudo dpkg -i sing-box_1.12.20_linux_amd64.deb
```

---

### Run App
```
python3 app.py
```
